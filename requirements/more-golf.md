# More Golf
The Masters tournament has been completed. The 2026 data should become archived which means there's no need to     
  request data from the API. Going forward, past years should be viewable in an "archive" navigation page. Next      
  year, in 2027, the Masters API should update to fetch the 2027 json. Until then, the data should be served         
  internally. No API requests should be made to the json file.                                                       
                                                                                                                     
  Further in 2026, there will be additional "golf" tournaments that are like the Master (with groups, 3 ball, and    
  leaderboard). However the data source for each golfer, round, and leaderboard will be different. A user will have  
  different selected golfers and 3 ball setups. I don't have the API information for those tournaments yet, but      
  let's setup the basis for "PGA Championship", "US Open", "British Open" to exist within a 2026 calendar yet.
