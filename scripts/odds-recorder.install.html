(If dragging doesn't work: right-click your bookmarks bar → Add page → paste the contents of scripts/odds-recorder.bookmarklet.txt as the URL.)

How to capture

  1. Open a sportsbook, log in, and go to the NFL futures / player-prop pages you want.
  2. Click the DZ Recorder bookmark — a panel appears top-right and starts recording.
  3. Navigate within the app (switch stat tabs / markets) so it fires fresh requests — the count climbs as each is captured. Avoid a full browser reload, which resets the recorder; if you do reload, just click the bookmark again.
  4. Hit Copy bundle (or Download) and send it back to Claude.

Suggested pages to visit per book (discovery pass):

• FanDuel — nc.sportsbook.fanduel.com/navigation/nfl?tab=player-props (and the futures tab)
• DraftKings — the NFL player-props / season-long futures pages
• theScore Bet — the 6 stat leaders pages (passing yds/tds, rushing yds, receiving yds/tds, team wins)

The recorder only reads JSON responses the page already loads and copies them to your clipboard. Nothing is sent anywhere automatically. This is a one-time discovery pass so Claude can see the real endpoint URLs + payload shapes before writing the parsers and the streamlined capture bookmarklet.
